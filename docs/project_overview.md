# Обзор проекта: Portfolio Website Frontend

## Описание

Фронтенд-приложение для личного сайта-портфолио разработчика. Проект создан с использованием современных веб-технологий: React 19, TypeScript, Vite и MobX для state management. Сайт демонстрирует навыки разработчика, проекты с GitHub, отзывы клиентов и предоставляет способы связи.

## Основные возможности

### Публичные страницы

- **Главная (Home):**
  - Hero-секция с информацией о разработчике
  - Секция навыков (Frontend, Backend, Tools, Soft Skills)
  - Секция проектов с GitHub интеграцией
  - Призыв к действию с ссылками на отзывы и контакты
- **Контакты (Contacts):**
  - Карточки контактов (Email, Telegram, GitHub)
  - Форма обратной связи с валидацией
- **Отзывы (Reviews):**
  - Список отзывов с рейтингом
  - Детальный просмотр отзыва

### Авторизованные пользователи

- **Заказ проекта (Order):** Форма оформления заказа с выбором типа проекта
- **Профиль (Profile):** Личный кабинет с историей проектов и отзывами
- **Аутентификация:** Регистрация и вход с JWT-токенами и HTTP-only cookies

### Администрирование

- **Админ-панель (AdminPanel):**
  - **Dashboard:** Статистика, сводка по проектам, последняя активность
  - **Пользователи:** Список с ролями и фильтрацией
  - **Проекты:** Управление статусами проектов
  - **Отзывы:** Модерация и удаление

### Общие возможности

- Адаптивный дизайн (Desktop, Tablet, Mobile)
- Мобильное меню (Drawer) с анимацией
- Профильное меню с быстрым доступом
- Cookie Consent баннер
- Политика конфиденциальности

## MobX Stores

Приложение использует 7 MobX stores для управления состоянием:

| Store                | Описание                                 |
| -------------------- | ---------------------------------------- |
| `authStore`          | Аутентификация, профиль, permissions     |
| `reviewsStore`       | CRUD отзывов, пагинация                  |
| `projectsStore`      | Проекты пользователя                     |
| `adminStore`         | Данные админ-панели                      |
| `githubStore`        | Репозитории и профиль GitHub             |
| `contactStore`       | Отправка сообщений через форму контактов |
| `cookieConsentStore` | Состояние согласия с cookie              |

## Система разрешений (Permissions)

| Разрешение          | Описание                               |
| ------------------- | -------------------------------------- |
| `projects:create`   | Создание проектов                      |
| `projects:read_own` | Чтение своих проектов                  |
| `projects:read_all` | Чтение всех проектов (Admin/Moderator) |
| `projects:manage`   | Управление проектами (Admin)           |
| `reviews:create`    | Создание отзывов                       |
| `reviews:delete`    | Удаление отзывов (Admin/Moderator)     |
| `users:read`        | Чтение списка пользователей            |
| `users:manage`      | Управление пользователями (Admin)      |
