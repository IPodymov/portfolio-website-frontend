# Обзор проекта: Portfolio Website Frontend

## Описание
Это фронтенд-приложение для личного сайта-портфолио. Проект разработан с использованием современных веб-технологий для обеспечения высокой производительности, отзывчивости и удобства поддержки. Сайт демонстрирует навыки разработчика, список проектов (с интеграцией GitHub), отзывы и предоставляет способы связи.

## Основные возможности
*   **Главная страница (Home):**
    *   Hero-секция с краткой информацией.
    *   Секция навыков (Frontend, Backend, Tools).
    *   Секция проектов: Динамическое отображение выбранных репозиториев с GitHub с кастомными описаниями.
*   **Проекты:** Интеграция с GitHub API для получения информации о репозиториях.
*   **Контакты и Заказ:** Формы обратной связи, интегрированные с Telegram (запрос username).
*   **Отзывы:** Страница с отзывами клиентов.
*   **Аутентификация:** Система регистрации и входа (использует JWT/Context).
*   **Адаптивность:** Полная поддержка мобильных устройств (Responsive Design), включая гамбургер-меню.
*   **UI/UX:** Использование Material UI для иконок и некоторых компонентов, кастомные CSS-стили.
*   **Legal:** Страница политики конфиденциальности и баннер согласия с Cookie.

## Технологический стек

### Core
*   **React 18+**: Библиотека для построения пользовательских интерфейсов.
*   **TypeScript**: Строгая типизация для повышения надежности кода.
*   **Vite**: Сборщик нового поколения, обеспечивающий быструю разработку и сборку.

### UI & Styling
*   **Material UI (@mui/material, @mui/icons-material)**: Библиотека компонентов и иконок.
*   **CSS Modules / Plain CSS**: Комбинированный подход к стилизации.
*   **Responsive Design**: Медиа-запросы для адаптации под мобильные устройства (breakpoints: 768px, 480px).

### State Management & Routing
*   **React Context API**: Управление глобальным состоянием (например, `AuthContext`).
*   **React Router DOM**: Маршрутизация на стороне клиента (SPA).

### API & Data
*   **Axios**: HTTP-клиент для взаимодействия с API.
*   **GitHub API**: Получение данных о проектах и профиле.

## Статус проекта
Проект находится в стадии активной разработки и поддержки. Реализован основной функционал, исправлены ошибки сборки (TS2769), оптимизирована мобильная версия.
