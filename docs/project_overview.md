# Обзор проекта: Portfolio Website Frontend

## Описание

Это фронтенд-приложение для личного сайта-портфолио. Проект разработан с использованием современных веб-технологий для обеспечения высокой производительности, отзывчивости и удобства поддержки. Сайт демонстрирует навыки разработчика, список проектов (с интеграцией GitHub), отзывы и предоставляет способы связи.

## Основные возможности

### Публичные страницы

- **Главная страница (Home):**
  - Hero-секция с краткой информацией.
  - Секция навыков (Frontend, Backend, Tools).
  - Секция проектов: Динамическое отображение выбранных репозиториев с GitHub с кастомными описаниями.
- **Контакты:** Форма обратной связи с валидацией.
- **Отзывы:** Страница с отзывами клиентов, детальный просмотр отзыва.

### Авторизованные пользователи

- **Заказ проекта:** Форма оформления заказа с выбором типа проекта.
- **Профиль:** Личный кабинет с историей проектов и возможностью оставить отзыв.
- **Аутентификация:** Регистрация и вход с JWT-токенами.

### Администрирование

- **Админ-панель:** Полнофункциональная панель управления:
  - **Dashboard:** Статистика (пользователи, проекты, отзывы), сводка по статусам проектов, последние проекты и отзывы.
  - **Пользователи:** Список всех пользователей, изменение ролей, удаление.
  - **Проекты:** Управление всеми проектами, изменение статусов.
  - **Отзывы:** Модерация отзывов, публикация/удаление.

### Общие возможности

- **Адаптивность:** Полная поддержка мобильных устройств (Responsive Design).
- **Мобильное меню:** Drawer-меню с плавной анимацией.
- **Профильное меню:** Выпадающее меню с быстрым доступом к профилю и админ-панели.
- **Legal:** Страница политики конфиденциальности и баннер согласия с Cookie.

## Технологический стек

### Core

- **React 18+**: Библиотека для построения пользовательских интерфейсов.
- **TypeScript**: Строгая типизация для повышения надежности кода.
- **Vite**: Сборщик нового поколения, обеспечивающий быструю разработку и сборку.

### UI & Styling

- **Material UI Icons (@mui/icons-material)**: Библиотека иконок (только иконки).
- **Pure CSS**: Кастомные CSS-стили без использования MUI-компонентов.
- **CSS Variables**: Централизованное управление цветами (`--color-primary`, `--color-accent`, `--color-text`).
- **Responsive Design**: Медиа-запросы (breakpoints: 768px для планшетов, 480px для мобильных).

### State Management & Routing

- **React Context API**: Управление глобальным состоянием аутентификации (`AuthContext`).
- **React Router DOM v6**: Маршрутизация на стороне клиента (SPA).
- **Custom Hooks**: Переиспользуемые хуки для форм и API-запросов.

### API & Data

- **Axios**: HTTP-клиент с интерсепторами для JWT-токенов.
- **GitHub API**: Получение данных о проектах и профиле.
- **Backend API**: REST API на `http://localhost:4000` (Express + TypeORM + PostgreSQL).

## Статус проекта

Проект находится в стадии активной разработки. Реализованный функционал:

- ✅ Аутентификация (регистрация, вход, JWT)
- ✅ Админ-панель с Dashboard
- ✅ CRUD для пользователей, проектов, отзывов
- ✅ Полностью адаптивный дизайн
- ✅ Интеграция с бэкендом API
- ✅ DRY-рефакторинг с переиспользуемыми компонентами
- ✅ Чистый CSS без MUI-компонентов (только иконки)
