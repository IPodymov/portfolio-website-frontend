# Руководство по установке и запуску

## Предварительные требования
Для работы с проектом вам понадобятся установленные:
*   **Node.js** (версии 18 или выше)
*   **npm** (обычно идет в комплекте с Node.js)

## Установка зависимостей
1.  Клонируйте репозиторий:
    ```bash
    git clone <repository-url>
    cd portfolio-website-front
    ```
2.  Установите зависимости проекта:
    ```bash
    npm install
    ```

## Запуск в режиме разработки
Для запуска локального сервера разработки с горячей перезагрузкой (HMR):
```bash
npm run dev
```
Приложение будет доступно по адресу: `http://localhost:5173` (или другой порт, указанный в консоли).

## Сборка для продакшена
Для создания оптимизированной сборки приложения:
```bash
npm run build
```
Результат сборки будет находиться в папке `dist/`.

### Предварительный просмотр сборки
Чтобы проверить, как будет работать собранное приложение локально:
```bash
npm run preview
```

## Линтинг
Для проверки кода на наличие ошибок и соответствие стилю:
```bash
npm run lint
```

## Устранение неполадок

### Ошибка TS2769 при сборке
Если вы сталкиваетесь с ошибкой `TS2769: No overload matches this call` в `Navbar.tsx`, убедитесь, что вы используете последнюю версию кода, где `NavLink` заменен на `Link` с использованием хука `useLocation` для определения активного состояния в компонентах Material UI.

### Проблемы с зависимостями
Если возникают странные ошибки при запуске, попробуйте очистить кэш и переустановить зависимости:
```bash
rm -rf node_modules package-lock.json
npm install
```
